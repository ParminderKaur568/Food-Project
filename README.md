## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic text editor or IDE (VS Code recommended)
- Git (optional, for cloning)

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/ParminderKaur568/Food-Project.git
   cd Food-Project
   ```

2. **Open in Browser**
   - Double-click `index.html` to open directly, OR
   - Use a local server for best results:
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Python 2
     python -m SimpleHTTPServer 8000
     
     # Using Node.js (if installed)
     npx http-server
     ```
   - Visit `http://localhost:8000` in your browser

3. **Customize (Optional)**
   - Edit `css/style.css` for colors, fonts, and styling
   - Modify `js/script.js` for functionality changes
   - Replace images in the `images/` folder with your own

## 📱 Responsive Design

This website is fully responsive and optimized for all screen sizes:

| Device | Breakpoint | Optimization |
|--------|-----------|--------------|
| **Desktop** | 1024px+ | Full layout with all features |
| **Tablet** | 768px - 1023px | Adjusted grid and navigation |
| **Mobile** | < 768px | Single column, mobile-friendly menu |
